## 1. Project Overview

This project aims to design, build and test a twin tilt-motor fixed-wing UAV.
The aircraft combines vertical take-off and landing capability with fixed-wing cruise efficiency.
Two independently tilting motors are used to provide vertical lift and forward thrust.

## 2. Design Objectives

The main objectives of this project are:
- To design a lightweight fixed-wing UAV with two independently tilting motors.
- To achieve vertical take-off and landing capability.
- To enable transition from hover mode to fixed-wing cruise mode.
- To maintain stable fixed-wing flight after transition.
- To investigate the effects of motor placement, centre of gravity, and tilt mechanism design on flight stability.
- To provide a modular platform for future control-system and aerodynamic improvements.
